More fun stuff
For some cutoff strategy $\tau(v)$, $$k(v) = \left\lfloor \frac{\tau(v)}{\overline{C}_B(v)} \right\rfloor$$
We take $\tau_{\text{coh}} = 10$ s
We define the memory coherence factor as: $$\Gamma_{\text{mem}}(v) = \frac{\overline{p}_B(v)}{1 - \left(1 - \overline{p}_B(v)\right)^{k(v)}} \sum_{n=1}^{k(v)} \left(1 - \overline{p}_B(v)\right)^{n-1} \exp\left(- \frac{n \overline{C}_B(v)}{\tau_{\text{coh}}}\right),$$ so that due to dephasing, the state of the first link is $$\rho = \frac{1 + \Gamma_{\text{mem}}(v)}{2} \left|\beta_{00}\right\rangle\!\! \left\langle \beta_{00} \right| + \frac{1 - \Gamma_{\text{mem}}(v)}{2} \left|\beta_{01}\right\rangle\!\! \left\langle \beta_{01} \right|$$ and the QBER is given by $$\operatorname{QBER}(v) = \frac{1 - \Gamma_{\text{mem}}(v)}{2}$$
Using the binary entropy, $$h(x) = -x \log_2(x) - (1-x) \log_2(1-x)$$ the secret key fraction is given by $$\operatorname{SKF}(v) = \max\left(0, 1 - h\left(\operatorname{QBER}(v)\right)\right).$$ The expected time per successful event is given by $$t_{\text{succ}}(v; \tau(v)) = \frac{\overline{C}_A(v)}{\overline{p}_A(v) \left(1 - \left(1 - \overline{p}_B(v)\right)^{k(v)}\right)} + \frac{\overline{C}_B(v)}{\overline{p}_B(v)}.$$
