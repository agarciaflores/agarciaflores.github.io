Lesson 1: Algebra

Arithmetic in \(\mathbb{Z}\)

All theorems and definitions in this section are adapted from [4].

Algebra originated from arithmetic and relies significantly on it. We will start our exploration of algebra by revisiting fundamental arithmetic concepts that serve as a framework for many of our mathematical investigations. I emphasize the fundamental structure and characteristics rather than computational techniques. However, the basic principles are ones that you are already familiar with.

The Division Algorithm

  1. Well-Ordering Axiom: Every nonempty subest of the set of nonnegative integers contains a smallest element.

Theorem 1.1 - The Division Algorithm Let \(a,b\in\mathbb{Z}\) with \(b>0\). Then \(\exists q, r\) \[ a = bq+r \text{ and } 0\leq r\leq b. \]
Let \(a\) and \(b\) be fixed integers with \(b>0\). Consider the set \(S := \left\{a-bx : x\in\mathbb{Z} \text{ and } a-bx \geq 0 \right\}\).
Choose \(x = -|a|\). Since \(b\) is a positive integer by hypothesis, we must have \[ \begin{align*} b &\geq 1, \\ b|a| &\geq |a| \\ &\geq -a; \\ a + b|a| &\geq 0, \\ a - b(-|a|) &\geq 0, \\ a - bx &\geq 0. \end{align*} \] Hence, \(a-bx\in S\) when \(x = -|a|\), \(\Rightarrow S\neq\varnothing\).
By the Well-Ordering Axiom, \(S\) has a smallest element—call it \(r\). Since \(r\in S\), we know that \(r\geq 0\) and \(\exists x \text{ s.t. } r = a-bx\), say \(x = q\). Thus, \(r = a - bq\) and \(r\geq 0\) \(\Leftrightarrow\) \(a = bq + r\) and \(r\geq 0\).
Now, suppose \(r\geq b\). Thus, \(r - b \geq 0\), so that \(0 \leq r - b = (a - bq) - b = a - b(q+1)\). Since \(0\leq a - b(q+1)\), \(a - b(q+1)\in S\) by definition of \(S\). Since \(b>0\), it is certainly true that \(r-b < r\). Thus, \(r - b = a - b(q+1) < r\), but \(r\) is the smallest value in \(S\); we arrive at a contradiction. Our assumption that \(r \geq b\) is false, and we conclude that \(r < b\). \(\therefore\), we have found \(q,r\in\mathbb{Z} \text{ s.t. } a = bq+r\) and \(0\leq r < b\).
Suppose \(\exists q^\prime, r^\prime\in\mathbb{Z} \text{ s.t. } a = bq^\prime + r^\prime\) and \(0\leq r^\prime < b\). Since \(a = bq + r\) and \(a = bq^\prime + r^\prime\), we have \(bq + r = bq^\prime + r^\prime\) so that \(b\left(q-q^\prime\right) = r^\prime - r\). Furthermore, \(0\leq r < b\) and \(0\leq r^\prime < b\). Multiplying the first inequality by \(-1\) (and reversing the direction), we obtain \(-b < -r \leq 0\) and \(0\leq r^\prime < b\). Adding these two inequalities tells us \[ \begin{gather*} -b < r^\prime - r < b, \\ -b < b\left(q-q^\prime\right) < b, \\ -1 < q - q^\prime < b. \end{gather*} \] Since \(q\) and \(q^\prime\) are both integers, \(q - q^\prime\) is an integer. The only integer between \(-1\) and \(1\) is zero, so \(q - q^\prime = 0 \Rightarrow q = q^\prime\). Since \(b\left(q-q^\prime\right) = r^\prime - r \Rightarrow 0 = r^\prime - r \Rightarrow r = r^\prime\). Thus the quotient and remainder are unique.
